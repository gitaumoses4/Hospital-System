# Java Hospital System Software

This is a complete Java Hospital Management software with all of the following features:

1. Login
![](https://github.com/gitaumoses4/Hospital-System/blob/master/screenshots/Screenshot%20from%202017-04-28%2016-22-54.png?raw=true)
    * Administrator Login
    * User Login
2. Manage inpatients
    * Patients admission
    * Bill management
    * Discharge patient
    * Medication
    * Doctor Consultation
3. Manage Outpatients
    * Patient Admission
    * Medication
    * Bill management
4. Doctor Management
5. Patient Bill Management
6. Medicine maintenance
7. Ward Mantenance
8. Department Maintenance
9. Appointments/ Channelling.
10. User Account management.
